#Steps to execute the Project:

Step 1: Open Podcast-summary-app folder
Step 2: Open cmd prompt
Step 3: Type streamlit run summ_app.py
Step 4: Now you will be redirected to Web Application.
Step 5: Open Listennotes.com in your browser
Step 6: Now select any podcasts of your wish.Then click option and click "use Api to fetch Json" option
Step 7: You will now see episode id . Copy the episode id
Step 8: Paste the episode id in the web application
Step 9: Now click submit
Step 10: It will take some time to process.
Step 11: You will receive Trancipt and Summary as an output
Step 12: You can also download the Trancipt and Summary by clicking download if required
